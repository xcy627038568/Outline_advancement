### 片段1-1-9·具体章节（第81章–第90章）

**第81章：三十万大军的阴影**
- 视角：朱高燧 / 耿炳文（南军主帅）
- 场景：北平布政使司衙门（现后勤总署） / 南军大营
- 核心事件：朝廷三十万平燕大军由老将耿炳文统帅，兵锋直指北平。北平城内人心惶惶，物价飞涨。主角面临着接手后勤以来的最大挑战：如何在被重兵围困的情况下，稳定军心并保证物资供应。
- 关键对话/心理：耿炳文看着北平地图冷笑：“竖子安敢造反？老夫围城三月，饿也饿死他们！”主角则在衙门里对着账本皱眉：“粮食只够两个月，必须立刻实行军事管制。”
- 推动剧情点：靖难之役的第一次大规模战役即将打响，生存压力拉满。
- 章末悬念：城内几家最大的粮商趁机囤积居奇，拒绝向燕军出售粮食。

**第82章：雷霆手腕平粮价**
- 视角：朱高燧
- 场景：北平最大粮行
- 核心事件：面对粮商的软抵抗，主角没有进行漫长的谈判，而是直接带着一队火枪手（初步装备自制火铳的护卫）踢开了粮行的大门。他以“通敌叛国”的罪名，当场查抄了三家最大的粮行。
- 关键对话/心理：粮商哭喊：“殿下，我们是正经买卖人啊！”主角一脚将其踢开：“国难当头，还想发国难财？全部充公！”
- 推动剧情点：用暴力手段迅速解决内部物资危机，展现战时后勤的铁血属性。
- 章末悬念：查抄虽然解了燃眉之急，但引起了北平城内其他商绅的极度恐慌和暗中串联。

**第83章：统购统销与票证制度**
- 视角：朱高燧
- 场景：北平后勤总署
- 核心事件：为了安抚民心并建立长效机制，主角颁布了《战时统购统销法》，废除金银交易，发行“燕军配给票”。所有物资由后勤署统一分配，凭票领取。同时，他向商绅承诺，战后凭票兑换双倍盐铁引。
- 关键对话/心理：“这不是抢劫，这是借！你们的命和钱现在都绑在燕军的战车上，车翻了，大家都得死！”
- 推动剧情点：在明朝强行建立了一套超越时代的战时计划经济体制。
- 章末悬念：一套复杂的票证系统需要庞大的算力支撑，传统的账房先生根本无法胜任。

**第84章：算盘与阿拉伯数字**
- 视角：落魄书生（现首席核算官） / 朱高燧
- 场景：后勤总署核算室
- 核心事件：面对堆积如山的账目，主角开办了“速成培训班”，强行向招募来的几百名落魄书生和落榜秀才推广阿拉伯数字、复式记账法和现代统计学。
- 关键对话/心理：书生们看着黑板上弯弯曲曲的数字：“这等番邦符号，有辱斯文啊！”主角一拍桌子：“学会的，月俸十两！学不会的，滚去城墙上搬石头！”
- 推动剧情点：引入现代财务管理工具，极大地提升了后勤系统的运转效率。
- 章末悬念：后勤运转刚刚走上正轨，前线却传来了战事不利的消息。

**第85章：真定之败与伤兵潮**
- 视角：朱能 / 青鸾
- 场景：真定前线 / 北平城内战地医院
- 核心事件：燕军先锋在真定遭遇耿炳文主力的伏击，伤亡惨重。大批伤兵被运回北平，原本的伤兵营瞬间爆满。残肢断臂和哀嚎声充斥着整个营地。
- 关键对话/心理：朱能满身是血地冲进医院：“殿下！前线扛不住了，耿老将的火铳队太猛了！”
- 推动剧情点：前线战局受挫，压力瞬间传导至后方医疗和军工系统。
- 章末悬念：主角看着伤口严重感染的士兵，发现之前的酒精和羊肠线已经无法应对如此大规模的创伤。

**第86章：战地医院的极限运转**
- 视角：朱高燧 / 青鸾
- 场景：北平战地医院（原大型寺庙改造）
- 核心事件：主角亲自下场，指挥医护人员进行残酷的“战场分诊”（Triage）：重伤且无救的放弃，优先抢救能重返战场的轻中度伤员。同时，他下令大规模截肢以防止坏死蔓延。
- 关键对话/心理：青鸾哭着拒绝给一个熟悉的士兵锯腿：“殿下，他还能救啊！”主角夺过锯子：“不锯腿，他今晚就会死于败血症！让开！”
- 推动剧情点：展现战争的残酷和医疗后勤在极限状态下的冷酷抉择。
- 章末悬念：大量伤兵被截肢后保住了性命，但失去了战斗力，燕军兵源告急。

**第87章：火器代差的碾压**
- 视角：朱高燧
- 场景：兵工厂（秘密据点）
- 核心事件：前线反馈，南军的火铳虽然落后但数量庞大。主角明白，必须在武器上形成绝对的代差。他利用这几天的时间，将兵工厂的所有产能集中，赶制出了第一批五百支真正意义上的“燧发枪”。
- 关键对话/心理：主角抚摸着冰冷的枪管：“耿炳文的火铳还得点火绳，下雨天就是烧火棍。这五百支燧发枪，我要教他什么是降维打击。”
- 推动剧情点：核心军工技术的又一次突破，燕军即将迎来火力反击。
- 章末悬念：主角决定亲自押送这批新式武器前往真定前线。

**第88章：三殿下的运输大队**
- 视角：朱高燧
- 场景：通往真定的官道
- 核心事件：主角带领着由老刘和伤愈归队老兵组成的五百人运输队，押运着燧发枪和大量颗粒火药，冒着南军游骑的骚扰，星夜兼程赶往真定。
- 关键对话/心理：“全速前进！哪怕跑死马，也要在南军发起总攻前把这批家伙送到父王手里！”
- 推动剧情点：主角首次离开大本营，深入一线战区。
- 章末悬念：运输队在距离真定大营还有三十里的地方，遭遇了南军精锐骑兵的拦截。

**第89章：燧发枪的首秀**
- 视角：南军骑兵将领 / 朱高燧
- 场景：真定外围荒野
- 核心事件：面对数倍于己的南军重装骑兵冲锋，主角没有选择结阵死守，而是命令五百运输队就地列成三排，直接掏出了尚未装配刺刀的燧发枪。
- 关键对话/心理：南军将领大笑：“一群运粮的辅兵，连长枪都没有，简直是找死！冲垮他们！”
- 推动剧情点：新式武器遭遇实战检验，生与死的一线之间。
- 章末悬念：骑兵冲至五十步，主角猛地拔出佩剑向下劈砍：“第一排，开火！”

**第90章：排队枪毙的艺术**
- 视角：朱高燧
- 场景：真定外围荒野
- 核心事件：燧发枪密集的弹雨瞬间撕碎了南军骑兵的重甲。三段击战术让火力连绵不绝，南军骑兵如割麦子般倒下，连主角的阵线都没摸到就全线溃败。
- 关键对话/心理：老刘看着手中冒烟的枪管，浑身颤抖：“殿下……这火器，简直是妖法！”主角冷酷地吹了吹枪口的硝烟：“这叫排队枪毙。走，去见父王。”
- 推动剧情点：主角携绝对武力突破封锁，不仅证明了新式武器的威力，更带着改变战局的钥匙抵达了靖难核心战场。
- 章末悬念：远处的真定大营方向，传来了震天的喊杀声，耿炳文的总攻开始了。
