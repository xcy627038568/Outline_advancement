# 故事世界观与全局大纲

> 本文件是全书的最高宪法，合并了整体设定、主线进展与历史时间线。任何章节的时间跨度和剧情方向不得违背此文件。

---

## 一、核心命题与世界规则

**1. 终极命题**：一个现代人穿越成大明朱高燧，如何在不造反的前提下，用商业手段成为大明真正的幕后主人？
- 权力、财富、家庭情感三者之间的张力。最终答案是“自由”——在海上建立自由港，摆脱皇权束缚。

**2. 穿越机制与知识边界**：
- 1399年5月穿越，灵魂替换，保留原身记忆和感情。
- 主角是现代商科/理工科普通人。懂经济规律（极高）、化学/机械基础（中高，能做肥皂/蒸馏酒/粗糙火药），但**绝对不懂**高精尖科技（如车床制造、电子学）。
- 任何超时代的科技引入必须遵循“试错→失败→改良→推广”的逻辑，禁止凭空造物。

**3. 三条核心冲突链**：
- **猜忌链**：朱棣防备造反 → 主角表现贪财无野心 → 财富过大引发更深猜忌 → 捐献资产化解。
- **资本与制度链**：主角工业/商业扩张 → 动了江南地主/传统手工业蛋糕 → 文官集团以“与民争利”发动金融战/弹劾 → 主角用硬通货反杀。
- **技术推进链**：现代技术落地引发社会结构变动（失业、城市化）→ 产生新阶层矛盾。

## 三、全书六卷主线规划与精确时间线（5000章卷纲设定）

### 第一卷：血火起步：乱世第一桶金与政治投机（第1—600章 | 1399—1402）
- **卷核心主题**：在战争极端环境下完成商业模式验证与资本原始积累。
- **主线目标**：在靖难绞肉机中存活并赚到30万两启动资金，拿到特许经营权。
- **世界格局变化**：建文削藩引发内战，北平化为绞肉机，最终大明权力中心从南京向北平转移。
- **主角成长**：边缘透明皇子 → 掌握军火后勤的实权王爷（赵王）。
- **关键冲突类型**：战争 / 后勤商战 / 皇权猜忌。
- **本卷高潮**：1. 计擒张昺谢贵，借医疗物资一战成名；2. 北平保卫战，李景隆50万大军围城下的后勤奇迹；3. 攻克南京，受封赵王。
- **本卷结局钩子**：靖难成功，但朱棣对主角庞大财力的猜忌刚刚开始，江南地主集团对“暴发户”的敌意初步显现，更大的朝堂风暴酝酿中。
- **精确历史节点**：
  - **1399.5-1399.6**：朱棣装疯卖傻，暗中筹备起兵，朝廷密使监视。（主角在偏院灯下黑发育）
  - **1399.7初**：计擒张昺谢贵，誓师起兵（夺九门）。
  - **1399.7中下旬**：燕军连克通州、蓟州等周边卫所。
  - **1399.8**：真定之战，大败耿炳文30万大军。
  - **1399.9**：朱棣北上大宁拉拢宁王，北平空虚。
  - **1399.10-1400.4**：李景隆50万大军围攻北平。极其惨烈的守城战。主角依靠后勤军火发家。
  - **1402.7**：朱棣攻克南京，即位永乐，主角封赵王。

### 第二卷：资本圈地：盛世财阀与商业帝国的崛起（第600—1500章 | 1403—约1415）
- **卷核心主题**：跨区域商业集团的建立及与传统农耕经济的初步碰撞。
- **主线目标**：建立覆盖基建、娱乐、金融的商业帝国，发行四海商票，化解文官弹劾。
- **世界格局变化**：大明重心北移（北京陪都），下西洋开启朝贡贸易，农业经济受到商业资本初步冲击。
- **主角成长**：普通藩王 → 掌控大明经济命脉的隐形财阀。
- **关键冲突类型**：朝堂权谋 / 金融商战 / 制度冲突。
- **本卷高潮**：1. 郑和下西洋背后的资本运作与博弈；2. 四海商票发行引发的挤兑与金融反杀；3. 徐皇后病逝前后的权力洗牌与危机。
- **本卷结局钩子**：徐皇后去世，主角失去最强保护伞；同时海外航线带来巨大利益，皇室与资本关于航海主导权及新大陆资源的争夺一触即发。
- **精确历史节点**：
  - **1403**：改北平为北京，升为陪都。
  - **1405**：郑和第一次下西洋。
  - **约1415年**：徐皇后去世（情感大转折，主角失去最强内院保护伞）。

### 第三卷：大航海时代：全球收割与技术破局（第1500—2500章 | 约1415—1425）
- **卷核心主题**：利用国家级航海工程实现全球资源收割与核心技术升级。
- **主线目标**：幕后控股皇家海运，引进美洲作物解决饥荒，研发燧发枪完成军事降维。
- **世界格局变化**：大明从内向型陆权帝国向海陆复合帝国转型，海外作物输入打破本土人口与土地资源瓶颈。
- **主角成长**：大明财阀 → 跨国远洋贸易寡头与技术垄断者。
- **关键冲突类型**：远洋开拓 / 军工研发 / 皇权夺嫡前期。
- **本卷高潮**：1. 正式迁都北京背后的基建利益瓜分；2. 三殿失火事件中的消防基建与政治洗牌；3. 美洲高产作物成功引进与燧发枪列装。
- **本卷结局钩子**：朱棣步入晚年，夺嫡之争白热化，汉王朱高煦手握重兵准备掀桌子，旧有军事力量与主角的新技术军队即将正面对撞。
- **精确历史节点**：
  - **1421.1**：正式迁都北京。
  - **1421**：三殿失火（主角借机大搞基建消防）。

### 第四卷：降维打击：工业萌芽与皇权交替（第2501—3300章 | 1425—约1430）
- **卷核心主题**：以早期工业化手段降维解决传统政治与军事危机。
- **主线目标**：研发蒸汽机原型，平定汉王叛乱，完成资本对国家机器的深度绑定。
- **世界格局变化**：旧军阀与传统卫所制度彻底没落，早期工业化产物（简易铁路、蒸汽动力）开始重塑大明权力逻辑。
- **主角成长**：技术垄断者 → 左右皇权更迭的“太上皇”级摄政王。
- **关键冲突类型**：武装叛乱 / 工业革命初现 / 皇权更迭。
- **本卷高潮**：1. 汉王朱高煦叛乱；2. 简易蒸汽轨道车运送神机营的降维打击；3. 朱棣驾崩与仁宗即位，主角以资产换取政治松绑。
- **本卷结局钩子**：新皇（朱高炽）虽是盟友，但全面工业化导致的大批传统手工业者失业及新老阶层撕裂问题全面爆发，社会动荡的火药桶被点燃。
- **精确历史节点**：
  - **1426前后**：朱高煦叛乱。主角用简易蒸汽轨道车运送神机营，降维打击。
  - **约1430年**：朱棣病逝。
  - **约1430年**：朱高炽（大哥）即位。

### 第五卷：蒸汽巨兽：全面工业化与阶层重塑（第3301—4100章 | 约1430—1460）
- **卷核心主题**：全面工业化推进下的社会结构重塑与新旧阶层博弈。
- **主线目标**：建成全国铁路网，确立内阁责任制雏形，打造铁甲舰，化解土木堡大危机。
- **世界格局变化**：大明完成初步工业化，形成资产阶级与无产阶级雏形，内阁与资本的权力开始压倒皇权。
- **主角成长**：大明实际统治者 → 现代政治制度奠基人与幕后退隐筹备者。
- **关键冲突类型**：阶级矛盾 / 国家生存危机 / 制度变革。
- **本卷高潮**：1. 京宁铁路通车引发的旧官僚利益反扑；2. 朱高炽逝世与朱瞻基即位的权力过渡；3. 魔改版土木堡大危机（商用飞艇与工程队的现代化救援）。
- **本卷结局钩子**：工业化大明已不可逆转，但皇权（朱瞻基）对主角这个“超越皇权的工业神明”产生了无法调和的终极恐惧，终局切割在即。
- **精确历史节点**：
  - **约1440年**：朱高炽去世，朱瞻基即位。
  - **约1449年**：土木堡大危机（本书改写为：失业驿卒与军阀炸毁铁路，困死皇帝。主角出动工程队和商用飞艇进行现代化救援）。
  - **约1450—1455年**：主角进入半退状态，交出部分实权。

### 第六卷：星辰大海：资本流亡与自由神话（第4101—5000章 | 约1460以后）
- **卷核心主题**：摆脱皇权体制束缚，确立纯粹的资本与契约自由领地。
- **主线目标**：剥离国内核心资产，建立海外自由港，完成从权力棋子到规则制定者的蜕变。
- **世界格局变化**：大明彻底沦为旧大陆霸主，而主角在海外建立的新型契约制自由港成为全球新灯塔，东西方大航海霸权初现端倪。
- **主角成长**：大明王爷 → 全球资本与工业的无冕之王（彻底超脱）。
- **关键冲突类型**：文明冲突 / 意识形态 / 终极自由博弈。
- **本卷高潮**：1. 与宣宗（朱瞻基）的终极摊牌与资产切割；2. 海外自由港的建国之战（抵御欧洲或大明保守派远征军）；3. 宣宗驾崩，主角彻底退隐成为时代传说。
- **本卷结局钩子**：（全书大结局留白）工业化的种子已撒满全球，主角的后代与大明的新一代将展开下一场更为宏大的文明竞赛。
- **精确历史节点**：
  - **约1460—1465年**：朱瞻基去世。主角彻底退隐，成为传说。
