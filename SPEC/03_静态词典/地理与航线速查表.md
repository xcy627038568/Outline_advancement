# 地理与航线速查表

> 本文件用于统一城内点位、城际路线、海贸航线与运输时间口径。
> 距离与时间一律按"写作可用的近似值"处理，不追求史学级精确。

---

## 一、使用原则

- 城内路程按"半个时辰、一个时辰"来写最顺
- 陆路时间优先受路况、季节、兵乱、关卡影响
- 海路时间优先受季风、停靠、补给、船型影响
- 若剧情需要更慢或更快，必须给原因

## 五、关键海贸点位

### 5.1 泉州

- 作用：主角东南货路、出海工匠、纸货与商票外放的重要节点
- 常见货：布、纸、瓷、药材、账册、船工
- 写作关键词：海商、税关、票号、舶货

### 5.2 广州

- 作用：更偏南货、糖、木材与杂色番货
- 常见货：糖、木料、硫黄、杂货、药材
- 写作关键词：南货、番商、转手、压价

### 5.3 旧港

- 作用：主角进入南洋网络的老跳板
- 常见货：香料、胡椒、药材、番布、航路消息
- 写作关键词：海盗、番邦、老海商、换舱

### 5.4 宁海自由港

- 作用：结算、换票、修船、裁争三位一体中枢
- 常见货：票据、银、香料、木材、修船材料、高值杂货
- 写作关键词：换票、修船、裁争

### 5.5 黑水湾

- 作用：安置朱高煦、收拢旧部、海防缓冲
- 常见货：粮、药、酒、布、低级军需
- 写作关键词：断供、旧部、暗线、海湾团练

## 六、路线时间写作口径

| 场景 | 建议写法 |
|------|----------|
| 城内送货 | 半日内、当日可回 |
| 百里内调货 | 1—3日 |
| 跨省陆运 | 10日以上 |
| 北平到南京 | 至少月余上下最稳 |
| 东南沿海到南洋 | 半月到一月，视风与停靠而定 |
| 黑水湾到宁海自由港 | 几日可达，适合写快船消息 |

## 七、不同路线适合运什么

| 路线类型 | 适合货类 | 不适合货类 |
|----------|----------|------------|
| 城内短驳 | 肥皂、酒、账册、小额现银 | 大宗粮草 |
| 陆路长运 | 金属件、药材、文书、高值货 | 低价值大体积日用品 |
| 河运 | 粮、布、盐、木材 | 极急军情 |
| 近海海运 | 瓷、丝、纸、糖、杂货 | 需要极快时效的机密文书 |
| 远洋海运 | 香料、高值杂货、珍稀作物 | 易腐、低值、短效货 |

## 八、第一卷重点地理提醒

- 前100章主舞台仍是 `赵王偏院 / 燕王妃处 / 世子府 / 西市酒坊巷 / 东城富户区 / 脚行集散口`

- 第一卷不要过早把视角切得太散，北平城内必须先写出熟悉感
